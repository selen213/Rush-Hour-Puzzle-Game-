# Rush Hour Puzzle Game 🚗

An object-oriented implementation of the classic Rush Hour puzzle game built using Smalltalk, focusing on clean OOP design, game logic, and interactive gameplay mechanics.

---

## Overview

This project implements a fully playable Rush Hour puzzle game using object-oriented programming principles in Smalltalk.

The system models:

- Vehicles
- Game boards
- Movement logic
- Collision handling
- Puzzle constraints
- Winning conditions

The project was designed with strong emphasis on:

- Object-oriented architecture
- Encapsulation
- Modularity
- Clean class hierarchy
- Game state management

---

## Features

- Interactive Rush Hour gameplay
- Vehicle movement system
- Collision detection
- Board validation
- Win-condition checking
- Dynamic game state updates
- Object-oriented game architecture
